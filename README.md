## Features

- **GLAZED CARGO Design System** — Premium frosted glass aesthetics
- **StoryBrand Framework** — Client-focused messaging
- **Fully Responsive** — Mobile-first design
- **SEO Optimized** — Schema.org markup, Open Graph tags
- **Interactive Elements** — Scroll animations, FAQ accordion, contact form
- **Production Ready** — 1,338 lines of clean HTML/CSS/JS

## Technology Stack

- Pure HTML5 + CSS3 + Vanilla JavaScript
- Single-file architecture (no build required)
- Google Fonts: Sora + DM Sans
- Lucide Icons
- Formspree for form handling

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. That's it! No build process needed.

## Before Going Live

Update the Formspree form ID in `index.html`:


## Design System

- **Primary Color:** `#FF6B35` (Orange — energy, logistics)
- **Typography:** Sora (display) + DM Sans (body)
- **Elevation:** Multi-layered shadows (z1–z5)
